# Expense Tracker

A Python-based Expense Tracker application that helps users manage income and expenses using SQLite database.

## Features

* Add Income
* Add Expenses
* View Transaction History
* Store Data in SQLite Database
* Automatic Balance Tracking

## Technologies Used

* Python
* SQLite3

## Project Structure

* TrackerSystem.py
* expense_tracker.db
